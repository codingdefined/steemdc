# SteemDC

A bot to display Steem profiles in Discord
![steemDC.PNG](https://xzor.xyz/ipfs/QmULUreEfeAGmtSpvABHdW4MhL5a7WPLdGucLMr97Qp3V4)

## Installation

- clone the repo
- run `npm install`
- copy config.js.dist to config.js and paste your bot token
- run `node bot`
